Getting Started with CertView Plugin

With CertView (certificate viewer) plugin you can view information about encrypted certificate files. Currently it supports viewing of X.509 certificates in the following formats: PEM (Base64 encoding), DER (encoded binary).

The plugin is based on the OpenSSL project and automates openssl commands, that parse and dump information from certificate files. Moreover, it does automatic detection of a file format, and supports most of the common certificate file formats. It asks user for password if the certificate file is password protected.

See Using Plugins for a description of basic work with plugin viewer, thumbnail loader and menu extension.
